# CSC-710-Final-Project-Media-Organizer

### Organizes photos and videos into YYYY/MMM (eg. "2026/Apr") folders based on whatever date we can pry out of their metadata. Optionally media can be renamed to MM_DD_YYYY_HHMMSS. Always previews before touching anything.

To run, download mo.py.

In a terminal window, run ```pip install hachoir``` (media metadata library)

Then, navigate (```cd dir```) to where mo.py is located and run ```python mo.py```
